# AGENTS.md — Codex Agent Guidance

This repository uses a standardized `.agents` directory for all agent-facing configuration.

## Canonical Agent Paths

- `.agents/README.md` — entry point for agent setup in this repo
- `.agents/PROJECT.md` — project-specific guidance for AI agents
- `.agents/skills/*/SKILL.md` — reusable task skills
- `.agents/settings.local.json` — local permission preferences

## Source of Truth

For agent workflow and automation, `.agents/*` is canonical.
